HTML

HTML（HyperText Markup Language）是一种用于创建和设计网页结构的标记语言。它由一系列的元素（elements）组成，这些元素通过标签（tags）定义，每个标签包裹着不同的内容，起到不同的作用。HTML 是 Web 开发中的基础，用于描述和定义页面的结构和内容。

基本结构

HTML 文档的基本结构由以下部分组成：


                <!DOCTYPE html>
                <html lang="en">
                <head>
                    <meta charset="UTF-8">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>My Web Page</title>
                </head>
                <body>
                    <h1>Hello, Prism.js!</h1>
                </body>
                </html>

标签与元素

HTML 中使用标签来创建元素，一些常见的标签和元素包括：


                    <!-- 段落 -->
                    <p>这是一个段落</p>
                    
                    <!-- 标题 -->
                    <h1>这是一级标题</h1>
                    
                    <!-- 超链接 -->
                    <a href="https://www.example.com">这是一个链接</a>
                    
                    <!-- 图像 -->
                    <img src="image.jpg" alt="图片描述">
                    
                    <!-- 列表 -->
                    <ul>
                        <li>无序列表项</li>
                        <li>无序列表项</li>
                    </ul>
                    
                    <ol>
                        <li>有序列表项</li>
                        <li>有序列表项</li>
                    </ol>

表单与输入

HTML 表单用于收集用户输入，包括输入框、按钮等元素：


                    <audio controls>
                        <source src="audio.mp3" type="audio/mp3">
                        Your browser does not support the audio tag.
                    </audio>
                    
                    <video width="320" height="240" controls>
                        <source src="video.mp4" type="video/mp4">
                        Your browser does not support the video tag.
                    </video>

form：定义表单

label：定义输入框标签

input：定义输入框，type 属性指定输入框类型，如文本、密码等。

学习心得

在这个数字化时代，网页设计作为一项重要的技能逐渐受到人们的关注。对于初学者而言，掌握网页设计可能是一项看似艰巨的任务。然而，正是在这样的学习过程中，一个人可以从无知到拥有技能，从迷茫到自信。

刚接触前端时，面对一系列陌生的标签、样式和脚本，我产生了浓厚的好奇心和兴趣。决心要深入学习前端开发，我开始踏上这个新领域的学习之旅。课堂学习只是前端知识的冰山一角，真正的理解和掌握需要通过实际操作。在课外时间，我积极参与小型项目，从实际问题中学到了许多解决方法，对前端的各个方面有了更深刻的认识。

虽然一开始有些吃力，但通过不断的实践和尝试，我发现网页设计并不是遥不可及的。现在，我能够创造简单而具有吸引力的页面，这让我感到非常满足和充实。这段初学者的心得告诉我，只要持续学习，每一步都有值得发现和体会的乐趣。
